== Description ==

Timeline Pro  is pure HTML & CSS timeline style grid for WordPress.

Display any post type using short-codes anywhere.

Plugin Details: http://kentothemes.com/items/social/kento-responsive-timeline-for-wordpress/



<strong>Plugin Features</strong><br />

* Use via shortcode.
* Display any post type.
* Custom color.
* Ajax load more post.
* Ajax load post comments.
* Ajax comments reply .
* Featured post marker.
* Social Share button.
* Click to zoom view for readers.
* Support all update browsers including IE  8
* Support any themes

 


== Installation ==

<strong>How to Display Kento Timeline on page?</strong><br />
To Display Ajax Gallery on page you need to add short code as following.<br />

`[timeline_pro posttype="post" bgcolor="#E3FBFF" maincolor="#ffc600"]`

 
parameter:<br />

    posttype - post type you want to display on timeline. (ex: post, portfolio)<br />
    bgcolor - background color for timeline area(ex: #E3FBFF ).<br />
    maincolor - main color of timeline(ex: #ffc600 ).<br />

You control shortcode parameter via admin page "Timeline Pro Settings"
